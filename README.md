# Demo: matplotlib

Installing Matplotlib
---

```sh
python3 -m pip install --user matplotlib
```
or

```sh
python3 -m pip install --user matplotlib --break-system-packages
```

Also need to install PyQt5:
```sh
python3 -m pip install --user PyQt5 --break-system-packages
```
